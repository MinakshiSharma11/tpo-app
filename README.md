# 🎓 TPO React App

This is a React-based web application designed to assist the **Training and Placement Office (TPO)** in managing
and displaying placement-related data efficiently.
---

## 📌 Project Overview

The **TPO App** is a single-page application (SPA) built using **React**. It provides a foundation to manage placement records,
student data, company details, drives, announcements, and more.
---

## ✨ Features

- ⚛️ Built with `create-react-app`
- 🧑‍🎓 Student and company data display
- 📅 Drives and event listing UI (in progress)
- 🔎 Clean and modern UI layout
- 🗃️ Component-based architecture for easy scaling

---

## 🛠️ Tech Stack

- **Frontend**: React (JavaScript)
- **Styling**: CSS / Tailwind / Bootstrap (optional)
- **State Management**: useState, useEffect (React Hooks)
- **Version Control**: Git + GitHub
